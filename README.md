# Study Planner System

AI 기반 맞춤형 학습 플래너 & 경쟁혁 학습 플랫폼

본 프로젝트는 학생의 성적 목표,학년,디플로마 선택,학습 가능 시간,학교 면학 여부 등을 기반으로
AI가 자동으로 최적의 학습 계획을 생성하는 스마트 학습 시스템입니다.

문제 풀이,랭킹, PVP 기능을 포함해 경쟁형 학습 플랫폼을 제공합니다.


# 주요 기능

## 1.AI 학습 플랜 생성
1등급 학생들의 실제 시간표 데이터를 기반으로
VAE,Transformer,GNN,DQN 모델을 활용해 맞춤형 학습 플랜을 제공합니다.

### 입력요소
- ET(방과후),EP(자율학습),면학/비면학 시간대
- 디플로마 선택
- 국,수,사,과,영,역,물,화,생 목표 등급(5등급제 기준)
- 과목별 목표 등급에 따라 학습시간 자동 배분

## 2.Todo 리스트
- '오늘의 할 일' 창에서 체크박스 표시

## 3.달력 플랜
- 하루 학습 완료 시 '완',미완료 시 '실' 표시
- 시즌 종료 후:
    - 목표달성 여부 확인
    - 부족한 과목 선택 -> 다음 시즌 플랜 보완 반영

## 4.플랜 생성 규칙
- 평일 + 학교 면학 시: [면학시간 - 과목 공부 시간]
- 평일 + 비면학 시: [과목 공부 시간]
- 주말: [과목 공부 시간]
- 목표 등급,디플로마에 따라 시간 자동 재조정

## 5.랭킹 시스템
- 모의고사 문제 기반 점수 부여
- 일일 랭킹,시즌 랭킹 지원
- 시즌 1위 상품 지급(미정)
- 랭킹에서 학생 이름 클릭 시 PVP 가능

  ## 6.PVP 시스템
  - 랜덤 문제 출제
  - 제한 시간: 약 5분

  ## 7.문제 시스템
  - 문제 풀이 탭에서 문제 풀이 가능
  - 문제 출처
      - 모의고사
      - 문제집(추가 예정)
   
### 배점
- 문제 당 2~5점
- 시즌마다 새 문제 추가

### 디플로마 점수 보정
| 디플로마 | 문과 과목 | 이과 과목 | 
|---------|------------|-----------|
|   이과   |    +10%   |   -10%    |
|   문과   |   원점수  |    +10%    |
|   예체   |     모든 과목 +10%     |

## 8.목표 시스템
- 국,수.,사,과,영,역,물,화,생 목표 등급 설정
- 시즌 종료 후 목표 달성 시 상품 제공(미정)

  ## 9.로그인 시스템
  - Google OAuth2 기반 인증(학교 학생만 가입 가능)
  - 로그인
      - 아이디/비밀번호
      - OAuth2

### 회원가입 입력 정보
- 아이디, 비밀번호
- 학년
- 디플로마
- 과목별 목표 등급
- 하루 학습 가능 시간
- 학교 면학 여부

  ## 10.면학 시간 설정
  - 평일
      - 학교 면학 선택: 공부시간 설정 불가
      - 비면학: 공부시간 설정 가능
 - 주말
     - 자유롭게 공부시간 설정

## 11. 학년별 플랜 구성

### 1학년
- 입력 과목: 국,수,사,통합과학,영,역
- 위 과목만 플랜 생성

### 2학년
- 선택 가능한 과목: 국,수,사,영,역,물,화,생
- 선택 과목만 플랜 생성

## 디플로마 종류

### 이과
- IT
- 공학
- 수학
- 물리
- 화학
- 생명과학
- IB(자연)

### 문과
- 인문학(문학/사학/철학)
- 사회과학
- 국제어문
- 경제경영
- IB(인문)

### 예체능
- 예술
- 체육
